# Roadmap

This roadmap is intentionally broad while IT Dashboard is in the planning stage.

## Phase 1 — Foundation

- Select the programming language and UI framework
- Define the application architecture
- Create basic window navigation
- Build a lightweight system overview
- Establish testing and documentation practices

## Phase 2 — Core Features

- Hardware information
- Performance monitoring
- Network information
- Storage information
- Services and event-log exploration

## Phase 3 — Expansion

- Additional troubleshooting tools
- Interface customization
- Accessibility improvements
- Plugin-system research and design

## Future Ideas

- Public plugin API
- Community plugins
- Additional device-management capabilities

Items may change based on testing, project scope, and community feedback.
