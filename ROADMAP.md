# Roadmap

This roadmap outlines the current development direction for IT Dashboard. Items may change based on testing, project scope, and community feedback.

## Completed Foundation

- [x] Select Python and PyWebView as the initial application stack.
- [x] Define the frontend and backend project structure.
- [x] Create the initial desktop application shell.
- [x] Establish a frontend-to-backend API bridge.
- [x] Add starter documentation, licensing, and dependency management.

## v0.3.0 — System Information Foundation

- [ ] Add a backend system-information module.
- [ ] Return structured system information through the API layer.
- [ ] Display operating-system, computer-name, architecture, Python-version, and application-version information.
- [ ] Create the first reusable dashboard information card.
- [ ] create basic CSS
