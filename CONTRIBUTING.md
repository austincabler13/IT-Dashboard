# Contributing

IT Dashboard is still in its planning and early-development stage.

Before contributing code:

1. Check whether the idea supports the project mission and design principles.
2. Open an issue to discuss large features or architectural changes.
3. Keep changes focused and explain what problem they solve.
4. Consider performance, usability, and maintainability.
5. Add or update documentation and tests when applicable.

Detailed development setup instructions will be added after the technology stack is selected.
